# 第一性原理推理技能

为 [Claude Code](https://claude.ai/code) 设计的结构化推理技能，通过交互式多阶段协议引导深度第一性原理分析。

**版本 2.0** — 完全重写：单文件自包含、交互式检查点、深浅模式、反模式警告、更丰富的参考资料。

---

## v2 新特性

- **自包含 SKILL.md**：所有模板、参考资料和工具均已内联，无需加载外部资源
- **交互式检查点**：每个阶段结束后暂停，等待用户审阅，不再一次性全部输出
- **深浅模式**：快速（2 阶段）、标准（4 阶段）、完整（5 阶段）——匹配问题复杂度
- **反模式警告**：明确说明何时**不该**使用第一性原理思维，以及常见翻车场景
- **元原理分类**：严格区分为 不可变定律 / 经验规律 / 启发式，并附诚实说明
- **深化案例研究**：时间线、被考虑的替代方案、濒死经历、具体数据
- **优先级排序**：ICE 框架用于排序重构的概念方案
- **精简偏误表**：精选 10 个偏误，附具体的第一性原理误用场景
- **整合压力测试**：6 个真正不同的测试维度，而非 5 个"换个环境"的变体

## 安装

1. 将 `first-principles-reasoning/` 文件夹复制到 Claude Code 的技能目录：
   - **个人级**：`~/.claude/skills/first-principles-reasoning/`
   - **项目级**：`your-project/.claude/skills/first-principles-reasoning/`
2. 重启 Claude Code（或新建会话），技能会被自动识别。

## 快速开始

用以下方式激活技能：

- *"从第一性原理分析这个问题"*
- *"把 X 的成本拆解到最基础的材料"*
- *"为什么这样做？从头重新思考一下。"*
- *"使用第一性原理推理技能。"*

---

## 五阶段协议

| 阶段 | 名称 | 做什么 | 检查点 |
|---|---|---|---|
| 1 | 明确元目标 | 剥离现有方案，表述最纯粹的功能性目标 | 与用户确认目标 |
| 2 | 挖掘并摧毁假设 | 列出假设、分类来源、摧毁非必要假设 | 与用户审阅判定 |
| 3 | 分解至基础要素 | 用 MECE 分解到不可再分的要素 | 与用户验证完整性 |
| 4 | 从基本真理重构 | 通过形态学矩阵生成新颖概念 | 与用户选择概念 |
| 5 | 证伪与迭代 | 设计最简实验，用 ICE 框架排序 | 呈现最终建议 |

每个检查点都会暂停等待用户反馈——不再一次性倾倒全部内容。

---

## 何时用 / 何时不用

**适用**：颠覆式创新、成本削减、创意瓶颈、"不可能"的问题。

**不适用**：常规问题、紧急决策、合规/安全问题、需要你缺乏的领域专业知识的难题。
