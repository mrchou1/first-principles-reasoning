# Logical Fallacies Quick Reference

| Fallacy | Description | Example | Correction |
|---|---|---|---|
| **Circular Reasoning** | The conclusion is assumed in the premise. | "This design is best because it's optimal." | "What independent evidence supports this?" |
| **False Dichotomy** | Presenting two options as the only ones. | "Either we keep the keyboard, or the device is unusable." | "What other options exist between these extremes?" |
| **Straw Man** | Misrepresenting an argument to refute it easily. | "So you think we should just ignore all safety regulations?" | "Can I restate their argument in its strongest form?" |
| **Appeal to Authority** | Citing an authority outside their expertise. | "A Nobel physicist says this business model will fail." | "What's their specific data and reasoning?" |
| **Appeal to Tradition** | Claiming something is right because it's old. | "We've always done it this way." | "Why was it originally done this way? Do those reasons still hold?" |
| **Post Hoc** | Assuming causation from sequence. | "After we added that feature, sales dropped, so it killed sales." | "What else changed at the same time?" |
| **Slippery Slope** | Claiming one step leads to an extreme chain. | "If we remove one button, soon there'll be no controls at all." | "What specific mechanism links each step?" |
| **Hasty Generalization** | Drawing a conclusion from too little data. | "Two people complained, so everyone hates it." | "What sample size would give me confidence? Do I have it?" |
| **No True Scotsman** | Redefining a category to exclude counterexamples. | "No real engineer would use that approach." | "What is the objective definition of the category?" |
| **Ad Hominem** | Attacking the person, not the argument. | "He's a dropout, so his design must be flawed." | "What's the argument itself, independent of the source?" |

## Self-Correction Protocol
After completing any reasoning step, ask:
1. "Have I committed any of these fallacies?"
2. "If an opponent were looking for a logical flaw, where would they strike?"
3. "Can I rewrite this argument to be airtight?"
