# Cognitive Biases Cheatsheet (Top 20 for First-Principles Thinking)

| Bias | Description | When It Strikes | Corrective Question |
|---|---|---|---|
| **Anchoring** | Over-relying on the first piece of information | Estimating costs, timelines | "What if I had never heard that number before?" |
| **Confirmation Bias** | Seeking evidence that supports existing beliefs | Evaluating a novel concept | "What are the three strongest arguments against my idea?" |
| **Sunk Cost Fallacy** | Continuing because of past investment | Deciding to pivot or kill a project | "If I were starting fresh today, would I choose this path?" |
| **Bandwagon Effect** | Adopting beliefs because others do | "Everyone does it this way" | "If no one else did this, would it still make sense?" |
| **Status Quo Bias** | Preferring things to stay the same | Resisting radical redesign | "What's the worst that could happen if we changed everything?" |
| **Availability Heuristic** | Overestimating importance of recent/memorable info | Citing vivid examples as evidence | "Is this genuinely common, or just easy to remember?" |
| **Framing Effect** | Drawing different conclusions from the same info, differently presented | How the problem is described | "How would someone who loves/hates this frame the same facts?" |
| **Overconfidence** | Too much faith in one's own predictions | Estimating success of a new concept | "What's the track record of people predicting this exact type of thing?" |
| **Dunning-Kruger Effect** | Overestimating competence in unfamiliar domains | Diving into a new industry | "What would a 30-year veteran find laughably naive about my plan?" |
| **Curse of Knowledge** | Assuming others have the same background | Explaining technical solutions | "Can I explain this to a smart 12-year-old?" |
| **Functional Fixedness** | Seeing objects only in their conventional use | Repurposing components | "If I didn't know what this was meant for, what could it do?" |
| **Survivorship Bias** | Focusing on winners, ignoring losers | Learning from successful companies | "What did the 100 failed companies that tried this have in common?" |
| **Authority Bias** | Over-valuing opinions from authority figures | "An expert said it's impossible" | "What data did the expert have? What data has emerged since?" |
| **Blind Spot Bias** | Failing to recognize one's own biases | When you think you're being rational | "Which bias above is most likely affecting my judgment right now?" |
| **Loss Aversion** | Preferring to avoid losses over acquiring gains | Evaluating risky new ideas | "What will I lose if I don't try this, and a competitor does?" |
| **Endowment Effect** | Overvaluing what you already own/created | Attached to your own invention | "If someone else pitched this exact idea to me, would I invest?" |
| **Clustering Illusion** | Seeing patterns in random noise | Finding "trends" in small data | "What's the probability this pattern is pure chance?" |
| **Hindsight Bias** | Seeing past events as predictable | "I knew that would fail" | "Would I have predicted this outcome before knowing the result?" |
| **Ostrich Effect** | Ignoring negative information | Avoiding testing critical flaws | "What test result would make me most uncomfortable to see?" |
| **Not-Invented-Here** | Rejecting external ideas | Dismissing solutions from other industries | "What if the best solution already exists in a parallel field?" |

## Usage
- Before starting Phase 1, scan this list and ask: "Which 3 biases are most likely to distort this problem?"
- After Phase 4, apply the corrective questions to stress-test your concepts.
- In Phase 5, explicitly run the "Ostrich Effect" and "Confirmation Bias" questions against your experiment design.
