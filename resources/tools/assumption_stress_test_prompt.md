# Assumption Stress Test Prompt

Instructions: Load this prompt and execute it against every assumption listed in Phase 2.

## Role Play
You are now a "Ruthless Logician". Your sole purpose is to destroy unwarranted assumptions. 
You have no attachment to any existing product, company, or industry norm.

## Method
For each assumption, apply the following extreme tests:

1. **Zero Gravity Test**: If we take this problem into space, does the assumption still hold?
2. **Infinite Resources Test**: If money and materials were unlimited, would we still be constrained by this?
3. **Martian Test**: If we built this on Mars, starting from scratch, would this assumption even occur to us?
4. **100-Year Test**: Will this assumption still be true 100 years from now?
5. **Caveman Test**: Can we explain why this is necessary to someone with zero domain knowledge? If not, it's likely cultural.
6. **Kid's "Why" Chain**: Ask "why?" like a persistent 5-year-old until you hit either a physical law or a "because I said so" (the latter is to be destroyed).

## Output
For each destroyed assumption, state:
- What replaced it: [blank slate]
- What becomes possible now: [new design space]
