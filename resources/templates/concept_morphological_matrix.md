# Morphological Matrix for Radical Recombination

> **Note**: This template is now fully integrated into SKILL.md (v2.0+). This file is kept as a standalone reference.

This tool helps you generate novel concepts by listing alternative implementations for each fundamental element, then combining them in unexpected ways.

## Example: Personal Transportation
| Element | A | B | C | D |
|---|---|---|---|---|
| Energy Source | Gasoline | Battery | Hydrogen Fuel Cell | Human Power |
| Propulsion | Internal Combustion | Electric Motor | Jet | Magnetic Levitation |
| Ground Contact | Wheels | Tracks | Legs | Hover (air cushion) |
| Control | Steering Wheel | Joystick | Neural | Autonomous |
| Enclosure | Metal Body | Composite Monocoque | Inflatable | None (exoskeleton) |

## How to Use
1. Fill the matrix with your fundamental elements (from Phase 3).
2. Generate concepts by picking one option from each row.
3. Force at least one "wild" combination (e.g., Human Power + Magnetic Levitation).
4. Check if the combination violates any fundamental law (Phase 5). If not, it's a valid concept.
